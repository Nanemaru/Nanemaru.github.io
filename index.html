LA GAME JAM GMTK 2024

Thème de la game Jam

- Le thème était “Built to scale”

- Pour coller à ce thème le jeu utilise deux idées. La première était un jeu de mot avec “scale” qui à la base signifiait “échelle” mais que nous avons interprété comme s’il voulait dire “escalader”. La deuxième idée, était tout simplement de faire un jeu avec le point de vue d’une souris ce qui nous fais voir le monde avec une échelle plus grande

MES GAME DOCUMENTS

Le premier jeu en Game document est un roguelike 2D en pixel art un peu dans le style de Dead Cells. On y contrôle ce qui est normalement un ennemi dans les jeux habituels. Le jeu se déroule en 5 niveaux et la particularité est, qu’à chaque fin de niveau, notre personnage va évoluer selon nos actions durant la partie. Par exemple, si avec notre Gobelin on a tué 80% des ennemis avec une attaque de mêlée, on évoluera avec un Gobelin à gourdin comme le montre l’arbre d’évolution ci- dessous.

Mon deuxième et dernier jeu en game document est un jeu de combat dans le style d’un Street Fighter mais avec une particularité en plus, le jeu est aussi un jeu de rythme. Mon idée est de faire en sorte que pour placer un combo il faut non seulement appuyer sur les bonnes touches, mais aussi se caler sur le rythme de la musique. La musique pourra être choisie avant le début du match. Si le joueur adverse veut contrer un combo, il faudra qu’il soit meilleur que son adversaire sur le rythme.
